## 1. 클래스

> 객체를 정의해 놓은 것, '객체의 설계도 또는 틀' 

하나의 역할만 하게 만드는 게 중요함!!

### 1. 기본 구성요소

* 데이터(변수) 

> 어떤 클래스를 구현하는지에 따라 달라짐

* 기능(메서드) 

> 해당 클래스가 해야 하는 일을 정의해 놓은 것

### 2. 정의 방법

```java
class Test {
    int number = 0;									// 멤버 변수
    String name = "";
    
    void SetName(String _name) { name = _name; }	// 메서드
    String GetName() { return name; }
    .
    .
    .
}
```

이런 식으로 정의할 수 있다.
